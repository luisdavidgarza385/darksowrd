BIENVENIDO A DARKSWORD

Para usar el panel de control necesitas correr el servidor local en tu PC.

1
Descarga el servidor Descargar ZIP
2
Extrae y ejecuta start.bat
3
Pone tu IP en Config Abrí el icono ⚙ arriba a la derecha

Guia de Instalacion

Requisitos

  • Python 3.7+ instalado
  • SPECTRAL X PRO corriendo en tu PC
  • Celular y PC en la misma WiFi

Paso 1: Descargar

Descarga los archivos del servidor desde GitHub:

Descargar darksword.zip

Paso 2: Instalar

Extrae el ZIP y abre la carpeta. Ejecuta:

pip install websocket_server

Paso 3: Correr el servidor

Doble click en start.bat o ejecuta:

python server.py

El servidor mostrara tu IP. Anotala.

Paso 4: Conectar

En la configuracion de este panel (icono ⚙), pone:

  • IP: La IP que mostro el servidor
  • Puerto: 8080

Hace click en "Guardar Conexion" y listo!
